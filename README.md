## ⚙️ Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/thekarthek/flutter_ecommerce_app.git
cd flutter_ecommerce_app
```

---

### 2️⃣ Install dependencies

```bash
flutter pub get
```

---

### 3️⃣ Run the app

```bash
flutter run
```

For web:

```bash
flutter run -d chrome
```
